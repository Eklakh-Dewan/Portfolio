# Portfolio — Eklakh Dewan

Personal portfolio site for Eklakh Dewan, AI Engineer / Data Science undergraduate at KPRIET.

**Live site:** https://eklakhdewan.com.np

## Stack
Plain HTML, CSS, and vanilla JavaScript — no build step, no framework. Deployed via GitHub Pages with a custom domain (see `CNAME`).

## Structure
```
index.html      → Home / hero / featured work
about.html      → Background, education, skills
projects.html   → Full project case studies (7 projects)
contact.html    → Contact form / links
style.css       → All styling (design tokens, layout, components)
script.js       → Nav toggle, typed hero animation, scroll reveals
assets/         → Images/icons
```

## Project status legend
Projects on the Projects page carry a status badge:
- **SHIPPED** — production or internship-deployed work, with measured outcomes
- **PROTOTYPE** — working end-to-end build, not production-hardened
